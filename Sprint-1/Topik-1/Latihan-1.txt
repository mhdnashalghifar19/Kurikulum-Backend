A. Cara mengaktifkan MS-Word

1.Klik star 
2.Pilih Program
3.Pilih Microsoft Word

B. Cara membuka lembaran kerja baru

1.Pilih menu File 
2.Klik New
3.Pilihlah Blank Document untuk membuat dokumen. atau
Klik Icon New pada toolbar standar atau
Tekan Ctrl+N pada keyboard

C. Cara menyimpan data

1.Pilih menu File 
2.Pilih Save As (untuk dokumen yang belum diberi nama)
3.Pada kotak Save In tentukan dimana akan disimpan (disket : flashdisk atau dalam hard disk)
4.Pada kotak Save Name beri nama dokumen
5.Pilih menu File klik Save atau tekan tombol Ctrl+S pada keyboard.
